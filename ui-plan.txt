📐 1. TECH STACK & ARSITEKTUR
--------------------------------------------------------------------------------
  Framework         : React 18+ (dengan Vite)
  Styling           : Tailwind CSS v3 + Headless UI
  Routing           : React Router v6
  State Management  : Zustand (global) + React Hook Form (form)
  Charts            : Recharts (Donut Chart di Dashboard)
  UI Icons          : Lucide React
  Toast/Notifikasi  : Sonner (toast ringan)
  Date Picker       : date-fns + custom DatePicker
  Export Excel      : xlsx (SheetJS) — generate .xlsx di client-side
  PWA               : vite-plugin-pwa (offline support)

  Pola Arsitektur:
    src/
      components/          → Shared UI (Button, Card, Modal, FAB, BottomNav, ...)
      features/            → Halaman & komponen spesifik fitur
        auth/              → LoginPage, RegisterPage
        profile/           → SetupPage, JobdescBuilder
        dashboard/         → HomePage, HeroChart, RecentActivities
        report/            → FormInputPage, CategoryPicker, DynamicFields
        history/           → HistoryPage, TimelineCard, SearchFilter, ExportButton
      hooks/               → useAuth, useJobdesc, useReport, usePWA
      stores/              → Zustand stores (authStore, jobdescStore, reportStore)
      lib/                 → utils, constants, api-client
      App.tsx
      main.tsx

🎨 2. DESIGN SYSTEM
--------------------------------------------------------------------------------

  🎯 Color Palette (Tailwind extended):
  ┌─────────────────────────────────────────────────────────────────┐
  │ Primary (Jobdesc Utama)    → emerald-500  #10B981 🟢            │
  │ Secondary (Improvement)    → blue-500     #3B82F6 🔵            │
  │ Accent (Help User)         → amber-500    #F59E0B 🟠            │
  │ Background                 → gray-50      #F9FAFB (light)       │
  │ Surface / Card             → white        #FFFFFF                │
  │ Text Primary               → gray-900     #111827                │
  │ Text Secondary             → gray-500     #6B7280                │
  │ Danger / Delete            → red-500      #EF4444                │
  └─────────────────────────────────────────────────────────────────┘

  🏷️ Typography:
    Font Family  : Inter (Google Fonts) — clean, modern, legible
    Heading      : font-bold, tracking-tight
      H1: text-2xl (24px)       — Judul halaman
      H2: text-xl (20px)        — Judul section
      H3: text-lg (18px)        — Judul card
    Body         : text-sm (14px) atau text-base (16px)
    Caption      : text-xs (12px) — meta info (tanggal, tag)

  📏 Spacing & Radius:
    Page Padding  : px-4 (16px) — konsisten di semua halaman mobile
    Card Padding  : p-4 (16px)
    Card Radius   : rounded-xl (12px) atau rounded-2xl (16px)
    Button Radius : rounded-lg (8px) atau rounded-full (pill)
    Gap antar-card: space-y-3 (12px)

  🌓 Shadows & Elevation:
    Card default   : shadow-sm border border-gray-100
    Card hover     : shadow-md (pada hover)
    FAB            : shadow-lg
    Bottom Nav     : shadow-[0_-2px_10px_rgba(0,0,0,0.05)]

    Mobile (< 768px)           Desktop (≥ 768px)
    ─────────────────────────────────────────────

    Full-width cards        →  max-w-4xl, centered, 2-kolom grid
    Filter BottomSheet      →  Filter Modal / Sidebar Panel
    DatePicker inline       →  DatePicker Popover / Dropdown
